# Form-Automation-System
Smart Form Automation System designed to streamline enterprise document workflows through automated data processing and report generation.


Libraries Used:-
pip install pandas openpyxl python-docx streamlit
Library Purpose:-
Pandas:-	Data processing and Excel sheet handling
OpenPyXL:-	Reading/writing Excel files
Python-Docx:-	Automated Word document generation


Testing:- Worked well for 250+ entries generating time ~35 seconds
